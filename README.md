# Advanced E-Commerce Management System

## Overview
An e-commerce platform developed using MERN stack with analytics and cloud deployment support.

## Features
- Authentication
- Product Management
- Cart & Checkout
- Inventory Management
- Analytics Dashboard
- Recommendation System

## Tech Stack
Frontend:
- React.js

Backend:
- Node.js
- Express.js

Database:
- MongoDB

Analytics:
- Python
- Pandas

Cloud:
- AWS EC2
- Docker
